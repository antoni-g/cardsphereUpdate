This setting only highlights new and updated packages when there is a set of saved packages for the exact package settings you have set on Cardsphere. You can turn this on or off at any time; any save notes settings and considers them only if this setting is selected.


With this selected, your saved packages will update as soon as you visit the Cardsphere send page.


The OK button displays on highlighted packages, allowing you to updated your saved packages and un-highlight them after review.


Since package values can change slightly as card indices update, this setting allows you set how much a package's price with otherwise saved contents has to change before it is highlighted again.
